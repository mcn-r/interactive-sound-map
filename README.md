# Sonic Garden — Interactive Sound Map

An interactive map interface designed for a living archive of environmental sound recordings.

This project explores how geographic space, sound, and interaction can merge into a minimal, immersive digital experience.

## Overview

Sonic Garden, created by Aziza Love, is part of the “From Sound to Soil” initiative — a growing archive of field recordings and installations that capture the sonic identity of landscapes.

This interface reimagines that archive as a spatial experience:
- a world map as the entry point
- ambient particle motion representing environmental presence
- interactive nodes tied to specific locations
- expandable content for sound playback and context

## Features

- Interactive world map with location-based nodes
- Custom particle system (HTML canvas) simulating atmospheric movement
- Pulsing “signal” markers indicating active sound locations
- Floating information card triggered by user interaction
- Responsive behavior optimized for mobile and desktop
- Fully custom HTML, CSS, and JavaScript (no external libraries)

## Interaction Design

The design emphasizes:
- subtle motion over distraction
- hierarchy (particles = ambient, pins = interactive)
- minimal UI to preserve immersion
- spatial storytelling through interaction

## Technologies Used

- HTML5
- CSS3
- JavaScript (Canvas API)
- Custom particle rendering system

## Purpose

This project was created as part of a creative technology portfolio, demonstrating the intersection of:

- interactive design
- generative visuals
- sound-based storytelling
- front-end development

## Future Improvements
